Project Overview

This project demonstrates the use of AWS services to create a secure and cost-effective backup system. I used AWS S3 to store backups, AWS KMS for encryption, AWS SNS for email notifications, and IAM for least-privilege access controls. Every Friday at 10:00 PM EST, the system automatically backs up all files from specified directories to an encrypted S3 bucket. Files are stored in the Deep Glacier storage class for cost optimization. The script performs incremental backups by comparing file hashes, uploading only new or modified files. Upon completion, an SNS email notification provides a summary of the backup status, size, and costs.

  • Drive Monitoring & Notification

    Alerts sent 5 minutes before backup to ensure drives are connected.

  • Change Detection (Hashing)

    Scans and hashes files to upload only changed content, saving bandwidth and storage.

  • Deep Glacier Storage

    Lifecycle rules transition snapshots to Deep Glacier for maximum cost efficiency.

  • Security (KMS & IAM)

    Encryption at rest and least-privilege roles for all components.
