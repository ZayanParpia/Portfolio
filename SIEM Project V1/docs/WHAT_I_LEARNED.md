\- \*\*Linux Dependencies\*\*

&#x20; - Understanding required packages and libraries for system tools

&#x20; - Managing installations and resolving dependency conflicts



\- \*\*Wazuh SIEM Platform\*\*

&#x20; - Deployment and configuration of Wazuh

&#x20; - Understanding how Wazuh collects, processes, and visualizes security events

&#x20; - Managing services using `systemctl`
