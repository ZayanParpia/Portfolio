\## Endpoint Validation



\### `auditd active on endpoint.png`

Shows that \*\*auditd\*\* is active on the Linux endpoint (The Ubuntu laptop), confirming that auditing and host level event logging are enabled.



\### `Endpoint Information.png`

Displays endpoint details such as the system hostname, operating system, and hardware information used to verify the endpoint.



\### `Ping Ubuntu Wazuh Server.png`

Shows local devices can communicate with Wazuh server



\### `Sysmon Running on endpoint.png`

Confirms that \*\*Sysmon\*\* is running on the endpoint, showing that detailed process and system activity monitoring is enabled.



\### `Wazuh Agnet Running on Endpoint.png`

Shows the Wazuh agent running on the endpoint, confirming that the host is properly configured with a Wazuh Agnet and sending logs to the Wazuh server.



\### `Ubuntu\_Laptop\_Overview.png`

Shows Ubuntu endpoint conencted to SIEM system
