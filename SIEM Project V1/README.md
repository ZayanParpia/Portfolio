# 🛡️ Home SIEM Lab

A hands-on cybersecurity home lab focused on Security Information and Event Management (SIEM), endpoint monitoring, log analysis, attack simulations, and SOC analyst workflows.

This project is designed to simulate real-world SOC operations while building practical blue team experience for portfolio and learning purposes.
