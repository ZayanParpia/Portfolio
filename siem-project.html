Project Overview

This project is a hands-on cybersecurity home lab focused on Security Information and Event Management (SIEM), endpoint monitoring, log analysis, attack simulations, and SOC analyst workflows.

It is designed to simulate real-world SOC operations while building practical blue team experience. The core of this environment involves setting up Wazuh SIEM and deploying agents to collect extensive endpoint telemetry using tools like Sysmon and Auditd.

Project Outline

"In this project, I built a hands-on cybersecurity home lab focused on Security Information and Event Management (SIEM), endpoint telemetry, and SOC security workflows. The core of this environment involves setting up Wazuh SIEM as our centralized manager and deploying agents to collect host logs. By installing Sysmon for Linux and auditd on an Ubuntu endpoint, I configured detailed telemetry to capture process creations, system changes, and file activity, ensuring extensive visibility for security monitoring and threat hunting."
