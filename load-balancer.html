Project Overview

This project demonstrates the implementation of a highly available Application Load Balancer (ALB) on AWS. It showcases how load balancing effectively distributes traffic across multiple EC2 instances to ensure application scalability and resilience.

The architecture utilizes a dedicated VPC, multiple Availability Zones, and public Subnets. Networking is managed through custom Route Tables and Network Access Control Lists (NACLs), while traffic is directed to specific Target Groups of EC2 instances based on health and availability.
